SQL: SELECT CHECKPATHID, CHECKPATHNAME FROM CHECKPATH WHERE OWNERID = ? AND UIOBJECT != 'ALERT' ORDER BY CHECKPATHNAME;
PLAN:
RETURN RESULTS TO STORED PROCEDURE
 ORDER BY (SORT)
  RECEIVE FROM ALL PARTITIONS
   SEND PARTITION RESULTS TO COORDINATOR
    INDEX SCAN of "CHECKPATH" using "CPOWNID"
     scan matches for (OWNERID = ?0), filter by (UIOBJECT <> 'ALERT')
